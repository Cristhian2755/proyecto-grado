### 2. **Modelos Refactorizados** ✓

Cambio de patrón: **MongoDB (constructor + save) → PostgreSQL (métodos estáticos)**

#### **User.js**
```javascript
// Métodos estáticos puros
User.findOne({ email })      // SELECT por email
User.findById(id)             // SELECT por id
User.create({...})            // INSERT usuario
```

#### **Project.js**
```javascript
Project.findAll()             // SELECT todos + JOIN usuarios
Project.findById(id)          // SELECT por id + JOIN
Project.create({...})         // INSERT proyecto
Project.update(id, {...})     // UPDATE proyecto
Project.delete(id)            // DELETE proyecto
```

#### **LineaTematica.js**
```javascript
LineaTematica.findAll()       // SELECT todas
LineaTematica.create({...})   // INSERT línea temática
```

#### **Evaluacion.js**
```javascript
Evaluacion.findByProject(id)  // SELECT por proyecto
Evaluacion.create({...})      // INSERT evaluación
Evaluacion.update(id, {...})  // UPDATE evaluación
```

#### **Notificacion.js**
```javascript
Notificacion.findByUser(id)   // SELECT por usuario
Notificacion.create({...})    // INSERT notificación
Notificacion.update(id, {...})// UPDATE notificación
Notificacion.delete(id)       // DELETE notificación
```

#### **Entrega.js**
```javascript
Entrega.findByProject(id)     // SELECT por proyecto
Entrega.create({...})         // INSERT entrega
Entrega.update(id, {...})     // UPDATE entrega
Entrega.delete(id)            // DELETE entrega
```

### 3. **Controllers Actualizados** ✓

#### **authController.js**
- `registerUser` → `User.create()` (sin constructor)
- `loginUser` → `User.findOne()` + `bcrypt.compare()`

#### **projectController.js**
- `createProject` → `Project.create()`
- `getProjects` → `Project.findAll()`
- `getProjectById` → `Project.findById()` (NUEVO)
- `reviewProject` → `Project.update()`
- `deleteProject` → `Project.delete()` (NUEVO)

---

### 4. **Rutas Configuradas** ✓

#### **authRoutes.js**
```
POST /api/auth/register     → Registrar usuario
POST /api/auth/login        → Login y obtener JWT
```

#### **projectRoutes.js**
```
GET    /api/projects        → Listar proyectos (requiere auth)
POST   /api/projects        → Crear proyecto (solo estudiantes)
GET    /api/projects/:id    → Obtener proyecto por ID (NUEVO)
PUT    /api/projects/review/:id → Revisar/actualizar (solo asesores)
DELETE /api/projects/:id    → Eliminar proyecto (NUEVO)
```

---

### 5. **Middleware Validado** ✓

- **authMiddleware.js** - Verifica JWT en `Authorization` header
- **roleMiddleware.js** - Valida rol del usuario (estudiante, asesor, etc.)
- **errorMiddleware.js** - Manejo centralizado de errores

## 📝 Notas Importantes

1. **Sin constructores innecesarios** - Los modelos ahora retornan objetos planos de PostgreSQL
2. **Métodos estáticos puros** - Todo es `await Model.method()`, sin `new` instances
3. **Parámetros seguros** - Usando `$1, $2, ...` para evitar SQL injection
4. **Migraciones ejecutadas** - Todas las tablas existen en la BD
5. **Dependencias instaladas** - npm install completado sin errores
